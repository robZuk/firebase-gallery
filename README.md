### Technologies
- React 17
- Firebase 9
- styled components 5
- material-ui 4
- framer-motion 4

### Features

- User Profile
- Adding photos
- Likes and comments on photos


### Env Variables
Create a .env file in then root and add the following

```
API_KEY="",
AUTH_DOMAIN="",
PROJECT_ID="",
STORAGE_BUCKET="",
MESSAGING_SENDER_ID="",
APP_ID=""
```

### Install Dependencies
```
npm install

```
### Run
```
npm start

```
